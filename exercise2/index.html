Explain why the result of ('b' + 'a' + + 'a' + 'a').toLowerCase() is banana.

  • 'b' + 'a' This concatenates the string "b" and "a" producing "ba".
  • While + 'a' attempts to convert the string "a" into a number. Because 'a' cannot be directly converted into a number, it returns NaN (Not-a-Number).
  • When 'ba' is concatenated with NaN, JavaScript converts the NaN to a string, resulting to 'baNaN'.
  • And then 'baNaN' + 'a' is concatenated and will result to 'baNaNa'.
  • The method .toLowerCase() converts the string into lowercase and will produce 'banana'.
